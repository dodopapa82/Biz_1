## 경쟁사 광고 성과 데이터 확보를 위한 데이터 요구사항 및 환경 준비 계획

CEO 지시에 따라 경쟁사 3곳의 광고 성과 데이터(CPM, CPC, 전환율 추정치) 확보에 필요한 구체적인 데이터 요구사항 목록을 정의하고, API 키/채널 ID 확보 시 즉시 분석할 수 있는 환경 준비 계획을 수립합니다.

### 1. 경쟁사 데이터 확보를 위한 데이터 요구사항 목록 (Competitor Data Requirements List)

경쟁사의 광고 성과를 효과적으로 비교 분석하기 위해 필요한 최소한의 데이터 항목 및 출처에 대한 요구사항입니다. 이 목록은 향후 데이터 요청 시 사용할 프레임워크가 됩니다.

| 구분 | 데이터 항목 | 세부 설명 및 측정 기준 | 예상 추정치 확보 난이도 | 비고 (데이터 출처 가정) |
| :--- | :--- | :--- | :--- | :--- |
| **광고 효율** | CPM (Cost Per Mille) | 광고 1,000회 노출당 비용. 도달 범위 및 광고 매체별 효율 비교에 필수. | 중 | 외부 광고 데이터 API 또는 직접 분석 필요 |
| **클릭 효율** | CPC (Cost Per Click) | 클릭당 비용. 잠재 고객 유입 비용 측정. | 중 | 외부 광고 데이터 API 또는 직접 분석 필요 |
| **전환 성과** | 전환율 추정치 | 광고 노출 대비 실제 구매/가입 등으로 이어진 비율 추정치. (매우 중요) | 상 | 간접 추론 및 시장 보고서 기반 추정 필요 |
| **콘텐츠 연관성** | 관련 콘텐츠 유형 | 경쟁사가 어떤 종류의 쇼츠(롱폼, 숏폼, 특정 주제)에 집중하고 있는지 파악. | 하 | 공개 채널 정보 분석 |
| **잠재 고객 세그먼트** | 타깃 인구통계 추정치 | 광고가 도달한 시청자의 예상 연령대, 성별, 관심사 추정치. | 상 | API 접근 또는 심층 리서치 필요 |

### 2. 데이터 분석 환경 준비 계획 (API Key & Channel ID Setup Plan)

실시간 및 정량적 분석을 위해 YouTube API 접근 권한 확보를 위한 구체적인 단계입니다.

**단계별 실행 계획:**

1.  **YouTube API 키 및 채널 ID 확보 (Priority 1):**
    *   **Action:** CEO/담당 부서에게 필요한 **YouTube API 키**와 **채널 ID**를 요청합니다. (→ `secretary` 에이전트 역할 수행)
    *   **목표:** 데이터 분석 파이프라인을 즉시 실행할 수 있는 환경 구축.

2.  **데이터 요구사항 명확화 및 연동 (Priority 2):**
    *   **Action:** 확보된 API 키/ID를 기반으로, 섹션 1에서 정의한 데이터 요구사항(CPM, CPC 등)에 맞춰 YouTube Data API의 가능한 항목을 매핑하는 **분석 프레임워크**를 개발합니다.
    *   **목표:** 요청된 데이터를 실제 API 응답과 연결할 수 있는 구체적인 코어 로직 설계.

3.  **경쟁사 데이터 연동 전략 (Priority 3):**
    *   **Action:** YouTube 데이터 외에 경쟁사 광고 성과를 확보하기 위한 **외부 데이터 소스(예: SEMrush, AdSpy 등) 연동 가능성**을 리서치하고, API 키 확보 후 통합 분석 방안을 수립합니다.
    *   **목표:** API 한계를 보완하여 요구되는 모든 데이터를 확보할 수 있는 다각적 전략 마련.
